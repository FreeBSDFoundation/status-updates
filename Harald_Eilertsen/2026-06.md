## **Reporting Period:** 2026-05-27 - 2026-06-02

### Project: Improve OpenJDK for FreeBSD
#### **Overall Status:** On Track
#### **Summary**

Finally got some reviews on the PR's I had for the upstreaming BSD port repo. One of them had to be withdrawn, and resulted in some cleanup in the out-of-tree BSD port (battleblow repo, see [PR#59](https://github.com/battleblow/jdk/pull/59)). For the other, upstream wants us to rework it so that it relies less on `#ifdef`'s by splitting the performance info modules into separate source files for macOS and BSD. That work has started, but it will probably take some iterations to find an acceptable way for this work. Trying to keep the changes as non-intrusive as possible, while still allowing room for the BSD specific code.

I also started to look into tests that is being skipped for various reasons on FreeBSD, and enabling the ones we can to increase test coverage. I plan to continue this work a bit on the side when time allows. Along with Kurt Miller, I've also started looking into backporting relevant FreeBSD fixes for jdk25 and 26 to earlier OpenJDK LTS versions. This is also work that will continue as time permits.
