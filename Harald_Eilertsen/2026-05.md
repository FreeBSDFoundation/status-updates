## **Reporting Period:** 2026-05-06 - 2026-05-12

### Project: Improve OpenJDK on FreeBSD
#### **Overall Status:** On Track
#### **Summary**

Submitted a somewhat more involved patch to the bsd-port upstreaming repo (still waiting for an upstream reviewer), and a couple of smaller PR's towards the mainline repo for some minor cleanups that eliminates some changes from the bsd-port. Also fixed an issue that cause compilation to fail on Aarch64 when the internal OpenJDK CPU feature flags were not in sync with the platform defined HWCAPS flags. Adopted the Linux approach that decouples these constants from each other.

Also based on a suggestion submitted via the FreeBSD bugtracker, fixed an issue in the javavmwrapper script that made it not work properly with OpenJDK versions higher than 21.

#### **Ongoing tasks**

- openjdk/bsd-port: [PR#9](https://github.com/openjdk/bsd-port/pull/9): Port hotspot/os/bsd/os_perf_bsd.cpp to BSD (Waiting for review)
