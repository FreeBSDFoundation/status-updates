## **Reporting Period:** 2026-01-21 - 2021-01-27

### Project: Vision Accessibility

Milestone 2: TUI high contrast theme.

#### **Overall Status:** On track
#### **Summary**

New high-contrast theme for TUI utilities in the BASE system.
[WIP repository](https://gitlab.com/alfix/freebsd-accessibility).

This week 4 themes were completed and a roadmap has been defined.

#### **Completed Tasks**

 - 4 themes:
   - No colors: black and white.
   - High Contrast: black, white, magenta, and yellow.
   - No yellow: white, black, red, and green.
   - No red: white, black, yellow, and blue.
 - A script to select a theme using a menu.

#### **Next Steps**

- Call for testing for the themes.
- Move theming features from bsddialog to libbsddialog.
- Improve theming: library and theme files.
- Release bsddialog 1.2, import base and port.
- Review to add theme menu to bsdinstall.

### Project: KDE installer dialogs
#### **Overall Status:** On Track for 15.1
#### **Summary**

The idea is: a user installs KDE at the end of bsdinstall through a series of dialog-based interfaces.
After rebooting, sddm starts and the user can log into the graphical environment.
[Repository](https://gitlab.com/alfix/kde-installer-dialogs).

The work was based on feedback received from the [testing call](https://lists.freebsd.org/archives/freebsd-desktop/2026-January/007438.html).

#### **Completed Tasks**

 - The script installs and set up: xorg, KDE Plasma, and sddm after a fresh installation.

#### **Features Delivered**

 - The _desktop_ script, tested with: VESA, SCFB, Intel, and AMD.
 - Added (untested) menu to select NVIDIA GPUs.

#### **Challenges**

 - Testing GPU drivers.
 - GPU auto-detection.

#### **Next Steps**

 - Add VirtualBox.
 - Add VMware.
 - Adapt the script to bsdinstall.
 - Review for CURRENT.

#### **Open Questions and Discussion**

https://gitlab.com/alfix/kde-installer-dialogs/-/issues
