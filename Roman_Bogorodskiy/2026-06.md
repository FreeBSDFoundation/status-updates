# 06 2026 Status Update for Roman Bogorodskiy

## **Reporting Period:** [2026-06-01] - [2026-06-07]

### Project: libvirt on FreeBSD
#### **Overall Status:** On Track
#### **Summary**

Updated the libvirt ports to the latest release.
Merged patches implementing querying the guest's network
interfaces information and filesystem information using the
guest agent.

Added
[support](https://lists.libvirt.org/archives/list/devel@lists.libvirt.org/thread/I67M7DDKTYOF7OVYLWAZ7PIIZSJWM7SU/)
for the `on_poweroff` and `on_reboot` actions, allowing e.g.
to configure a VM to shut down on reboot or reboot on shut down etc.

Fixed a couple of
[bug](https://lists.libvirt.org/archives/list/devel@lists.libvirt.org/thread/MIH3DVVWDHDU2VMM4HM63VIOUFW6TXQK/)
discovered with the new guest agent code.

Implemented a few more guest agent based
[APIs](https://lists.libvirt.org/archives/list/devel@lists.libvirt.org/thread/GSJHSGRRCCUXCTHR7PO3FPHUZGWA4TZ2/),
such as:
 - getting and setting guest's system time,
 - setting user's password in the guest,
 - managing SSH keys in the guest.

#### **Completed Tasks**

 * Ports update to the new release.
 * Filesystems and network interfaces information querying via guest agent.
