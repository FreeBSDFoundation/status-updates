### Project: libvirt on FreeBSD
#### **Overall Status:** On Track
#### **Summary**

This week was fully devoted to the CI effort.

vanilla images downloaded from https://download.freebsd.org/ftp/snapshots/VM-IMAGES/.

The idea is to bootstrap the image and then run the same
bastille-based suite I'm already running as a part of the existing CI setup.

This seems to be working, need to polish a few things and think how to plug it into
the existing CI.

Couple of issues noticed so far:

 * Unlike my existing UFS setups, for this approach I'm using ZFS
   images so I can prepare the image on the Linux host. Some
   of the storage tests are failing because of that, will need
   to take a look.
