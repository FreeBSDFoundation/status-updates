## **Reporting Period:** 2026-05-19 - 2026-05-26

Still fairly busy with other things but did find time for pkgbasify maintenance work

### **Work Completed**

- pkgbasify maintenance
   - Added a prompt to allow users to fetch pkg from the latest ports repo if the available version is older than 2.7
		- This is a workaround for the fact that pkgbasify requires pkg 2.7+ but only pkg 2.6 is available in current quarterly ports branch.
		- This workaround can be removed after the next quarterly ports branch is cut
		- The workaround is implemented such that it will not cause any problems if people continue to use pkgbasify versions with the workaround in the future.
   - Fixed some minor out of date information in the docs
   - Tagged a v1.1.0 release
