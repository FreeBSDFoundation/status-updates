## **Reporting Period:** March 2, 2026 – March 9, 2026

### **Project:** Sylve — GUI for Managing Bhyve, Jails, and More on FreeBSD

#### **Summary**

Prepared an early release candidate (RC) version of Sylve and began broader community testing. I also launched the Sylve landing page and documentation site at https://sylve.io and https://sylve.io/docs. In addition, I announced a call for testing on r/freebsd and Discord to gather early user feedback.

To make testing easier, I introduced "tip" releases on GitHub (https://github.com/AlchemillaHQ/Sylve/releases/tag/tip), which automatically provide builds from the latest commit that passes CI. So people don't have to install heavy language run times and mess with compiling to just get up and running fast.

#### **Completed Tasks**

* Launched the Sylve landing page and documentation site (https://sylve.io)
* Announced a call for testing on r/freebsd and the Sylve Discord community
* Introduced "tip" releases that automatically publish builds from the latest CI-passing commit
* Continued improving and expanding documentation and setup guides

#### Blockers

No major blockers at the moment. Currently focusing on gathering user feedback and identifying issues during the testing phase.
